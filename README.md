# A Curvature Identity for the Fibonacci Digital-Root Cycle Modulo 9

This repository contains the source files and compiled PDF for the paper:

**A Curvature Identity for the Fibonacci Digital-Root Cycle Modulo 9**  
*Guy Rinzema, July 2025*

## Overview

We classify the sum of squared second differences (“digital curvature”) for every 24-periodic orbit generated by the Fibonacci recurrence modulo 9. The curvature sum takes exactly three values: 648, 972, and 1296. The maximal value, 1296 = 9 × F₁₂, is attained only by the classical Fibonacci digital-root cycle and its rotations.

## Files

- `main.tex`: LaTeX source file of the paper
- `Fibonacci_Mod9_Curvature_Identity.pdf`: Final compiled version of the paper
- `verify_curvature.py`: Script used to verify all possible 24-periodic seeds modulo 9
